🔧 実行方法

1. 事前準備

1-1. Python環境の準備

Windows WSL 上でPythonをインストールし、本フォルダー内の「型式一覧」以下にある requirements.txt を利用してモジュールをインストールします。

仮想環境 venv を先に入れて実行します。

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

1-2. 最新ファイルのダウンロード

上記の国土交通省のサイトより、取り込まれていないファイルを「型式一覧」フォルダ以下にダウンロードし、年代別のフォルダを作成してファイルを配置します。

フォルダ名には和年月を記載してください。

2. CSVファイルの作成

WSLで「型式一覧」フォルダに移動し、以下のコマンドを実行します。

python excel_consolidator.py

実行後、CSVファイルが出力されます。

3. JavaScriptファイルへのコンバート

WSLで「Co2排出量計算」フォルダに移動し、以下のコマンドを実行します。

python convert_csv_to_js.py

実行後、JavaScriptファイルが「型式一覧」フォルダに作成されます。

⚠️ 注意事項

LPGに付いて
LPGは意図して省略しています。
給油時レシート記載は L 単位になり、計算するうえではこれを kg に変換の必要性があり、
その変換は、気温によってプロパン/ブタンの比率が変わりL⇒Kg変換に必要な係数が変動するため
