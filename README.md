# CO2排出量計算ツール

車両の走行距離から二酸化炭素（CO2）排出量を計算するWebアプリケーションです。

## ⚠️最初に：単位には気をつけましょう 
- メートル <> キロメートル 
- キログラム <> トン 


## 📋 機能 

### タイプA: 実燃費から計算 (calc_method:0)
実際の燃費データを使用してCO2排出量を計算します。

**入力項目:**
- 実燃費 (km/L)
- 燃料種別（ガソリン 2.32 / 軽油 2.58 / LPG 3.00）
- 走行距離 (km)

**計算式:**
```
CO2排出量 (kg-CO2) = 走行距離(km) ÷ 実燃費(km/L) × 燃料係数 ÷ 1000
```

### タイプB: 型式別燃費から計算 (calc_method:1)
国土交通省の型式別燃費データを使用してCO2排出量を計算します。

**入力項目:**
- モード（WLTC / JC08 / 10・15 / JH25 / JH15）※未選択の場合は自動検索
- 型式（例: 3BA-NRE210H）
- 燃料種別（ガソリン 2.32 / 軽油 2.58 / LPG 3.00）
- 走行距離 (km)

**計算式:**
```
CO2排出量 (kg-CO2) = 走行距離 ÷ 燃費値 × 燃料係数 ÷ 1000 × モード係数
```

**モード係数:**
| モード | 係数 |
| ------ | ---- |
| WLTC   | 1.0  |
| JC08   | 0.9  |
| 10・15 | 0.8  |
| JH25   | 1.0  |
| JH15   | 0.9  |

**検索順序:** WLTC → JC08 → 10・15 → JH25 → JH15

## 🚀 使い方

### 基本的な使い方

1. `index.html` をブラウザで開く（ダブルクリック）
2. 必要な情報を入力
3. 「計算する」ボタンをクリック
4. CO2排出量が表示されます

### 型式の入力形式

**形式:** `英数3文字-英数1文字以上`

**正しい例:**
- `3BA-NRE210H`
- `DBA-ZVW50`
- `DAA-NHP10`

**注意:**
- 全角で入力しても自動的に半角に変換されます
- 全角ハイフン（－、ー）も自動的に半角に変換されます

   ※ローカルで実行するためにCSVをJSに変換しています。

```bash
python convert_csv_to_js.py
```

## 📊 CSVファイルの形式

CSVファイルは以下の形式である必要があります：

```csv
フォルダ名,ファイル名,車名,通称名,型式,燃費値（km/L）
令和2年3月,001337994_WLTC_ガソリン乗用車（普通・小型）.xls,トヨタ,カローラ スポーツ,3BA-NRE210H,16.4
令和2年3月,001337994_WLTC_ガソリン乗用車（普通・小型）.xls,スズキ,ジムニー,3BA-JB74W,15.0
...
```

**必須項目列:**
- `型式` - 車両の型式指定番号
- `燃費値（km/L）` - カタログ燃費値

## 🧪 使用例

### 例1: 実燃費から計算

**入力:**
- 実燃費: 15.5 km/L
- 燃料種別: ガソリン (2.32)
- 走行距離: 10000 km

**計算:**
```
10000 ÷ 15.5 × 2.32 ÷ 1000 = 1.497 t-CO2
```

### 例2: 型式別燃費から計算（モード指定）

**入力:**
- モード: WLTC
- 型式: 3BA-NRE210H
- 燃料種別: ガソリン (2.32)
- 走行距離: 10000 km

**計算:**
```
燃費値（WLTC）: 15.8 km/L
10000 ÷ 15.8 × 2.32 ÷ 1000 × 1.0 = 1.468 t-CO2
```

### 例3: 型式別燃費から計算（自動検索）

**入力:**
- モード: 自動検索（未選択）
- 型式: 3BA-NRE210H
- 燃料種別: ガソリン (2.32)
- 走行距離: 10000 km

**動作:**
- WLTCモードで型式を検索 → 見つかった場合、そのデータを使用
- 見つからない場合、次のモード（JC08）を検索
- 以降、順番に検索（10・15 → JH25 → JH15）
